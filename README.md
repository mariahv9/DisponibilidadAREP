## Clonacion del repositorio 

Para clonar el proyecto usar el siguiente comando:

```
https://github.com/mariahv9/DisponibilidadAREP.git
```

### Ejecucion App

```
cd Disponibilidad
mvn package
```

### Crear javadoc

```
mvn javadoc:javadoc
```
